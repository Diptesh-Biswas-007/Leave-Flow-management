# LeaveFlow 🟠

A full-stack Employee Leave Management Portal built with Node.js, Express, 
and vanilla JavaScript — no frameworks, no database dependencies.

## Features
- 🔐 Secure login & signup with session-based authentication
- 📋 Apply, track, and filter leave requests
- 🔔 Notifications & approval workflows
- 👥 Team calendar view
- 📊 Leave usage reports & analytics
- ⚙️ Profile & settings management
- 📱 Fully responsive — works on mobile, tablet & desktop

## Tech Stack
- **Backend** — Node.js, Express, express-session
- **Frontend** — Vanilla JS (SPA), Bootstrap 5, Bootstrap Icons
- **Auth** — Session-based (no JWT, no database required)
- **Fonts** — Syne, DM Sans (Google Fonts)

## Getting Started
git clone https://github.com/yourusername/leaveflow.git
cd leaveflow
npm install
node server.js

Then open http://localhost:3000
